# Pixel Game

A pixelated isometric portfolio playground built with vanilla HTML5 canvas.

## Run

Open `index.html` in a browser.

## Controls

- Arrow keys to move the player around the map

## Features

- Isometric/2.5D tile world with a diamond grid
- Orthographic-style projection (no perspective distortion)
- Depth from terrain layering, object shadows, and directional lighting tones
- Consistent pixel scale, palette, outlines, and crisp rendering
- Tall objects (trees/building/signs) rendered with depth-aware draw order
- A playable character sprite with isometric movement and collision against water/obstacles
